# Twemoji

Replace Google Noto Emoji with Twemoji

## Description

Replace `/system/fonts/NotoColorEmoji.ttf` with the font you got from [JoeBlakeB/ttf-twemoji](https://github.com/JoeBlakeB/ttf-twemoji)

It has been confirmed to work on LineageOS 22.2, 23.0 with KernelSU, and KernelSU Next.
